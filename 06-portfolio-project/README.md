## Features

- Light/Dark theme toggle
- Sections for About, Experience, Projects, and Contact
- Social/profile links (LinkedIn, GitHub, LeetCode)

## Run Locally

1. Open this folder in VS Code.
2. Open `index.html` in your browser (or use Live Server).

# 01 – DOM Interactions

A hands-on exercise page exploring core browser DOM APIs in Vanilla JavaScript.

## Live Demo
[Open ↗](https://adityashahhhwal.github.io/learning-2025/01-dom/)

## What It Covers
- Selecting elements with `getElementById`, `querySelector`, and `querySelectorAll`
- Responding to user events with `addEventListener`
- Toggling and manipulating CSS classes dynamically
- Reading and updating `innerHTML` / `textContent`
- Traversing the DOM (parent, child, sibling relationships)

## Built With
- HTML5
- CSS3
- Vanilla JavaScript (ES6)

## Key Takeaway
Understanding how the browser represents a page as a tree of nodes, and how JavaScript can read and mutate that tree in response to user actions, is the foundation every frontend feature is built on.

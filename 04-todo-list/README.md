# 04 – To-Do List

A clean, interactive to-do list app that lets users manage tasks in the browser.

## Live Demo
[Open ↗](https://adityashahhhwal.github.io/learning-2025/04-todo-list/)

## Features
- Add new tasks via text input
- Mark tasks complete / incomplete with a checkbox
- Delete individual tasks
- Task count updates dynamically
- Persists tasks in `localStorage` so they survive page refreshes

## Built With
- HTML5
- CSS3
- Vanilla JavaScript (ES6)

## What I Practiced
- **Dynamic rendering** — generating list items from a JS array and keeping the DOM in sync
- **`localStorage`** — serialising and deserialising state with `JSON.stringify` / `JSON.parse`
- **Event delegation** — attaching a single listener to the list container instead of each item
- **Data-driven UI** — treating the JS array as the single source of truth and re-rendering from it

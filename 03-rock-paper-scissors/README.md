# 03 – Rock Paper Scissors

A classic Rock Paper Scissors game playable against the computer in the browser.

## Live Demo
[Open ↗](https://adityashahhhwal.github.io/learning-2025/03-rock-paper-scissors/)

## Features
- Player vs. Computer with randomised computer choice
- Win / lose / draw detection with result message
- Running score tracker across multiple rounds
- Animated choice feedback with icons

## Built With
- HTML5
- CSS3
- Vanilla JavaScript (ES6)

## What I Practiced
- **Conditional logic** — encoding game rules cleanly with `if/else` / lookup tables
- **Randomness** — using `Math.random()` to simulate the computer's choice
- **Score state** — persisting and displaying round-by-round results
- **DOM updates** — reflecting game state changes in the UI without page reloads
